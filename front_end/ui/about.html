About Us

Many file sharing and education sites cost money and are not open source. This often results in locked features and slower development. We want to provide a free, open-source, secure, in-depth education sharing platform where users can help each other in academia or even contribute code to the site.

Objectives
○ Create an application to manage and share files
○ Crowdsourcing student generated files
○ Manually allow any users to upload documents to user created groups
○ Allow tags, ratings, and comments of documents ○ Allow students to ask and answer questions from peers

We will deliver an application accessible by web that will fulfill the objectives stated above. The technology stack we are looking to utilize is as follows:
● NodeJS on the Backend.
● PostgreSQL for the Database.
● HTML/CSS/JS for the Frontend.
